FAQ

Frequently Asked Questions

Get answers to common questions about Smart Link Intelligence and how it can transform your link management strategy.

What is Smart Link Intelligence?

Smart Link Intelligence is a next-generation link management platform that uses AI to analyze user behavior and route visitors to the most appropriate destination based on their engagement level, creating personalized user experiences.

How does the DDoS protection work?

Our 5-layer DDoS protection system automatically detects and mitigates attacks in real-time. It includes rate limiting, burst detection, IP filtering, and automatic link protection that can temporarily disable links under attack and restore them when safe.

Can I customize behavior rules for my links?

Yes! You can create custom behavior rules with your own thresholds for visitor classification. Set custom time windows, visit counts, and engagement levels to define when users are considered curious, interested, or highly engaged.

Is there an API for developers?

Absolutely! We provide a comprehensive RESTful API with full documentation, webhooks for real-time events, and easy integration options for developers and businesses looking to integrate smart link functionality into their applications.
