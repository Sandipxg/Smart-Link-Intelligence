Creating Smart Links

Smart Link Intelligence offers three powerful redirect modes to suit different use cases.

Redirect Modes Explained

1. Standard Redirect All Users

What It Does:

Direct, immediate redirect to your destination URL with no intermediate pages or delays. Perfect for simple link shortening and tracking.

Best For:

  • General link sharing
  • Social media posts
  • Email campaigns
  • Basic click tracking

How It Works:

  1. Visitor clicks your short link
  2. System logs the visit (IP, device, browser, location, etc.)
  3. Visitor immediately redirected to destination URL
  4. Analytics updated in real-time

2. Progressive Redirect Elite & Elite Pro

What It Does:

Intelligent routing system that sends visitors to different URLs based on their engagement level and behavior patterns.

Configuration Fields:

Returning Visitor URL:

  • URL for visitors who return within configured time window
  • Example: Send returning visitors to special offer page
  • Triggered when user visits again within "Returning Window"

CTA URL (Highly Engaged):

  • URL for highly engaged users who click multiple times
  • Example: Send engaged users to premium product or sign-up page
  • Triggered when user clicks multiple times (based on behavior rule)

3. Password Protected Elite & Elite Pro

What It Does:

Adds authentication layer requiring visitors to enter password before accessing destination URL.

Configuration:

Creating Custom Rules

Rule Configuration:

1. Rule Name:

  • Descriptive name for your rule
  • Examples: "E-commerce Rule", "Blog Rule", "SaaS Funnel"

2. Returning Window (Hours):

  • Time window to consider visitor as "returning"
  • Short (12-24h): Fast-paced campaigns, flash sales
  • Medium (48-72h): Standard marketing campaigns
  • Long (168h/1 week): Long-term nurturing, education

3. Interested Threshold (Visits):

  • Number of visits needed to be "Interested"
  • Low (1-2): Quick qualification, impulse products
  • High (3-5): Complex products, high-consideration

4. Engaged Threshold (Clicks):

  • Total clicks needed to be "Highly Engaged"
  • Low (2-3): Quick conversion paths
  • High (5-10): Complex sales cycles, enterprise
Example Rules:

E-commerce Flash Sale: 12h window, 1 visit, 2 clicks

SaaS Free Trial: 72h window, 2 visits, 4 clicks

Educational Content: 168h window, 3 visits, 5 clicks

Elite Pro Exclusive

What Is DDoS Protection?

Automatically detects and mitigates malicious traffic patterns that could:

  • Skew analytics with fake clicks
  • Overwhelm destination servers
  • Waste bandwidth and resources
  • Compromise data integrity

Dashboard Metrics:

  • Protected Links: Links with active protection
  • Under Protection: Links in heightened security mode
  • Disabled Links: Temporarily disabled due to attacks
  • Total Events: Cumulative security incidents

Protection Levels

Level Status Action Impact
Level 0 Normal Operation No threats detected No visitor impact
Level 1-2 Monitoring Increased Enhanced analysis, rate limiting Minimal impact
Level 3 Captcha Required Visitors must complete CAPTCHA Bots blocked
Level 4 Temporary Disable Link disabled for 1 hour Auto-recovery
Level 5 Manual Recovery Link disabled indefinitely Requires manual review

How It Works:

  • Rate Limiting: Monitors clicks per IP per time period
  • Burst Detection: Identifies sudden traffic spikes
  • Pattern Analysis: Detects bot-like behaviors
  • IP Reputation: Cross-references known malicious IPs
  • Behavioral Analysis: Identifies non-human patterns
